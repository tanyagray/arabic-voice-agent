- Example: Māori (not Maori), whānau (not whanau), kōrero (not korero)
- Ensure all Te Reo text uses proper macrons for accurate text-to-speech

- Macrons indicate long vowels that are held longer
- Wh is pronounced like 'f' in most dialects
- Ng is pronounced like in 'singer', not 'finger'
- R is a soft rolled 'r'
